System Overview

Purpose

Define a controllable aquaponics system where feed composition acts as the primary input variable influencing plant growth via nutrient cycling.

---

Core Flow

Feed → Fish → Waste → Bacteria → Nutrients → Plants → Water Return

---

Subsystems

1. Fish Tank

- Species: Experimental use of piranhas
- Function:
  - Convert feed into ammonia-rich waste
  - Act as biological input processor

---

2. Mechanical Filtration

- Removes:
  - Solid waste
  - Uneaten feed
- Prevents:
  - Anaerobic buildup
  - Toxic spikes

---

3. Biofilter

- Hosts nitrifying bacteria:
  - Nitrosomonas (NH₃ → NO₂⁻)
  - Nitrobacter (NO₂⁻ → NO₃⁻)
- Critical for:
  - Detoxifying ammonia
  - Producing plant-available nitrate

---

4. Grow Beds

- Media or hydroponic roots
- Absorb:
  - Nitrate
  - Trace minerals
- Act as:
  - Secondary biofilter
  - Nutrient sink

---

5. Recirculation Loop

- Pumps water continuously
- Maintains:
  - Oxygenation
  - Nutrient distribution

---

Control Points

Variable| Control Method
Feed rate| Timed / adaptive feeding
Protein %| Feed selection
pH| Buffering agents
Flow rate| Pump control
Oxygen levels| Aeration

---

Design Philosophy

- Treat system as a closed-loop biochemical processor
- Focus on input-output relationships, not just growth
- Enable repeatable experiments

---

Constraints

- Fish stress affects output stability
- Bacterial colony requires time to stabilize
- Overfeeding causes system collapse
